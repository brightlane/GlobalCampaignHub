# GlobalCampaignHub

Affiliate marketing site featuring 70 verified merchants across 100 languages, powered by automated blog generation via GitHub Actions.

## Sites

- [GlobalCampaignHub](https://brightlane.github.io/GlobalCampaignHub/)
- [Omni-Hub](https://brightlane.github.io/omni-hub/)
- [Verified Merchant Directory](https://brightlane.github.io/verified-merchant-directory/)

## How it works

Blog posts are automatically generated every 30 minutes using GitHub Models (gpt-4o-mini), translated into 100 languages, and published to GitHub Pages. Affiliate links are tracked through LinkConnector.

## Stack

- GitHub Actions — automated blog pipeline
- GitHub Models — AI content generation
- GitHub Pages — static site hosting
- LinkConnector — affiliate tracking
